Tensorflow.js

Tensorflow.js is a Javascript implementation of the popular Tensorflow library used for many Machine Learning applications. It enables us to use off-the-shelf JavaScript models or convert Python TensorFlow models. Moreover, we can retrain models for our data, which is what I have done in this project.
